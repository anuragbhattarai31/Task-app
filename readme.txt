My-Task App

Description
-----------
My-Task App is a simple and efficient task management application designed to help users keep track of their daily to-do items. It allows users to add, edit, and delete tasks with ease. The app showcases a lightweight frontend with a robust backend, hosted using modern cloud services.


Features
--------
- Add tasks to your list.
- Edit task details.
- Delete tasks no longer needed.
- Persistent storage for tasks.

Tech Stack
----------
- Frontend: React
- Backend: AWS Lambda
- Database: DynamoDB
- Hosting: AWS Amplify

How to Run Locally
------------------
1. Clone this repository:
   git clone https://github.com/anuragbhattarai31/Task-app

2. Navigate to the project directory:
   cd Task-app

3. Install dependencies:
   npm install

4. Start the development server:
   npm start

5. Open your browser and go to http://localhost:3000.
